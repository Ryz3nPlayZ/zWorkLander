# zWorkLander

A modern, responsive landing page built with React, TypeScript, and Tailwind CSS v4.

## Features

- **Modern Tech Stack**: Built with React 19, TypeScript 5.7, and Vite 6.2
- **Styling**: Tailwind CSS v4 for utility-first styling
- **Animations**: GSAP for smooth, performant animations
- **Icons**: Lucide React for beautiful, consistent icons
- **Routing**: React Router DOM v7 for client-side routing
- **Type Safety**: Full TypeScript support for enhanced developer experience

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Install dependencies
npm install
```

### Development

```bash
# Start development server
npm run dev
```

The application will be available at `http://localhost:5173`

### Build

```bash
# Build for production
npm run build
```

### Preview

```bash
# Preview production build
npm run preview
```

## Deployment

### Netlify

The project is configured for Netlify deployment. Push to your repository and connect to Netlify for automatic deployments.

### Vercel

The project includes `vercel.json` for Vercel deployment. Push to your repository and import to Vercel.
